# AI Canteen Preorder Optimization Environment

## 🚀 Problem
Canteens face inefficiencies in handling multiple food orders, leading to delays and food waste.

## 💡 Solution
This project simulates a real-world canteen preorder system where an AI agent learns to optimize order handling.

## ⚙️ Environment Design
- State: time, pending orders, completed orders, waste
- Actions:
  - Prepare order
  - Batch prepare
  - Reject order

## 🎯 Objective
Maximize efficiency while minimizing delay and waste.

## 🧠 Tasks
- Easy: Handle single order
- Medium: Manage multiple orders
- Hard: Optimize profit and reduce waste
